## Useage

github page的问题，`index.html` 加入了以下代码

```html
  <meta
      http-equiv="Content-Security-Policy"
      content="default-src *; style-src 'self' http://* 'unsafe-inline'; script-src 'self' http://* 'unsafe-inline' 'unsafe-eval'"
    />
```

如果本地双击打开看效果注释上边代码就可以了。
